\section{Notation and Preliminaries}

\subsection{Fields}

Fields of size $q$ are denoted as $\FFF_q$ or simply $\FFF$.

\subsection{Reed-Solomon codes}

We use the following notation:
\begin{itemize}
    \item $RS[\FFF, S, \rho]$: Reed-Solomon code over the field $\FFF$ with evaluation domain $S$ and rate $\rho$.
    \item $\deg(f)$: degree of the polynomial $f$.
\end{itemize}
